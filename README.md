## Development Blog
* v26.3.38 : "In this update, we’ve introduced several new features and squashed a few bugs to improve your experience. You can now edit your tweets without altering their original timestamps, and we've added the convenience of pasting images directly into the tweet box, which functions just like a standard upload. For media viewing, we've enabled swipe navigation so you can easily switch between images. We've also refined how threads work: replying within the thread view now attaches your response directly to the root tweet, and clicking "thread" on any deeply nested reply will now correctly fetch the entire conversation from the very top. Finally, we've resolved an issue to ensure the date filtering feature works exactly as intended."
